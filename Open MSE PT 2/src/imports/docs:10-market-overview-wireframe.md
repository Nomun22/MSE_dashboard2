# Market Overview Wireframe

## Page purpose
Give the user a one-screen summary of the Mongolian Stock Exchange market situation.

## Main sections
1. Top header
2. KPI row
3. Index and market summary row
4. Trading activity row
5. Disclosure and report row
6. News / announcement row

## 1. Top header
Contains:
- page title: Market Overview
- market date
- language toggle
- search bar
- quick navigation to Securities Overview

## 2. KPI row
Cards:
- TOP-20 Index
- MSE ALL
- Market Capitalization / Зах зээлийн үнэлгээ
- Daily Trading Value / Арилжааны үнийн дүн

## 3. Index and market summary row
Left:
- TOP-20 summary card
  - current value
  - daily change
  - short note

Right:
- Market summary card
  - total trading value
  - notable market comment
  - selected highlights

## 4. Trading activity row
Left:
- market activity table preview
Right:
- trading summary text block

## 5. Disclosure and report row
Left:
- latest disclosures table
  - symbol
  - date
  - company name
  - type

Right:
- latest financial reports table
  - symbol
  - date
  - company name
  - type

## 6. News / announcement row
Left:
- board decisions and meeting notices
Right:
- market news highlights / announcements

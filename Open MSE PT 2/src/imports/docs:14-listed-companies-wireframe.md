# Listed Companies Page Wireframe

## Page purpose
Let the user browse, search, and inspect all listed companies on MSE.

## Main sections
1. Top header
2. Filter and search bar
3. Main company table
4. Detail drawer / side panel
5. Footer note

## 1. Top header
Contains:
- page title: Listed Companies
- result count
- language toggle
- search field
- export button (UI only for now)

## 2. Filter and search bar
Contains:
- search by symbol
- search by company name
- sort option
- reset filters action

## 3. Main company table
Columns:
- No.
- Symbol
- Company Name

Behavior:
- sticky table header
- row hover state
- click row to open detail drawer
- search results update table

## 4. Detail drawer / side panel
Shows:
- selected symbol
- selected company name
- category: Listed Company
- source section: open.mse.mn/securities
- future links:
  - financial information
  - governance
  - shareholder meetings
  - disclosures

## 5. Footer note
Shows:
- source URL
- collection date
- methodology note

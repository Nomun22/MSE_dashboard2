# Securities Overview Wireframe

## Page purpose
Give the user a structured overview of all securities categories available on MSE.

## Main sections
1. Top header
2. Category KPI row
3. Category cards row
4. Preview tables row
5. Cross-link row

## 1. Top header
Contains:
- page title: Securities Overview
- search bar
- language toggle
- quick link to Market Overview

## 2. Category KPI row
Cards:
- Total Listed Companies
- Total Debt Instruments
- Total ABS Listings
- Total Investment Funds

## 3. Category cards row
4 large cards:
- Listed Companies
- Debt Instruments
- Asset-backed Securities
- Investment Funds

Each card shows:
- category name
- total count
- short description
- button / link to open full page

## 4. Preview tables row
Top left:
- Listed Companies preview table
  - Symbol
  - Company Name

Top right:
- Debt Instruments preview table
  - Symbol
  - Security Name
  - Maturity Date

Bottom left:
- ABS preview table
  - Symbol
  - Security Name
  - Maturity Date

Bottom right:
- Investment Funds preview table
  - Symbol
  - Fund Name

## 5. Cross-link row
Contains:
- link to Market Overview
- link to Disclosures / Reports
- link to Security Detail
- methodology / source note

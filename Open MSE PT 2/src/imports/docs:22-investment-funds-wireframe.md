# Investment Funds Page Wireframe

## Page purpose
Let the user browse and inspect all investment funds listed on MSE.

## Main sections
1. Top header
2. Filter and search bar
3. Main fund table
4. Detail drawer / side panel
5. Footer note

## 1. Top header
Contains:
- page title: Investment Funds
- result count
- language toggle
- search field
- export button (UI only for now)

## 2. Filter and search bar
Contains:
- search by symbol
- search by fund name
- sort by symbol
- reset filters action

## 3. Main fund table
Columns:
- No.
- Symbol
- Fund Name

Behavior:
- sticky table header
- sort by symbol
- click row to open detail drawer
- search updates table results

## 4. Detail drawer / side panel
Shows:
- selected symbol
- selected fund name
- category: Investment Fund
- source section: open.mse.mn/securities
- future related links

## 5. Footer note
Shows:
- source URL
- collection date
- methodology note

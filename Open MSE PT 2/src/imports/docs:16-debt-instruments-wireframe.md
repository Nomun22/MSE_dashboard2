# Debt Instruments Page Wireframe

## Page purpose
Let the user browse, search, sort, and inspect all debt instruments listed on MSE.

## Main sections
1. Top header
2. Filter and sort bar
3. Main debt table
4. Detail drawer / side panel
5. Data quality note area

## 1. Top header
Contains:
- page title: Debt Instruments
- result count
- language toggle
- search field
- export button (UI only for now)

## 2. Filter and sort bar
Contains:
- search by symbol
- search by instrument name
- sort by maturity date
- sort by symbol
- reset filters action

## 3. Main debt table
Columns:
- No.
- Symbol
- Security Name
- Maturity Date

Behavior:
- sticky header
- sort by maturity date
- sort by symbol
- click row to open detail drawer
- search updates table results

## 4. Detail drawer / side panel
Shows:
- selected symbol
- selected debt instrument name
- category: Debt Instrument
- maturity date
- source section: open.mse.mn/securities
- future related links

## 5. Data quality note area
Shows:
- suspicious record warning
- duplicate / tranche note
- methodology note
